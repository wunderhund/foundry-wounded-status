# Wounded Status Effect

A lightweight FoundryVTT module that adds a custom "Wounded" status effect to your game.

## Features

- Adds a "Wounded" status icon to the token status effects menu
- Simple overlay effect for marking injured characters
- Compatible with FoundryVTT v13+

## Installation

### Method 1: Manifest URL
1. In Foundry VTT, go to the "Add-on Modules" tab
2. Click "Install Module"
3. Paste this manifest URL:
   ```
   https://raw.githubusercontent.com/wunderhund/foundry-wounded-status/main/module.json
   ```
4. Click "Install"

### Method 2: Manual Installation
1. Download the latest release
2. Extract to your Foundry `Data/modules/` directory
3. Restart Foundry VTT

## Usage

1. Enable the module in your world settings
2. Right-click any token on the canvas
3. Select the "Wounded" status effect from the status effects menu
4. The wounded icon will appear on your token
